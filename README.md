# ChessGame

## Description

simple chess game made using c++ 

## How to run

```bash

mkdir build
cd build
cmake  -G "Unix Makefiles" ..

make

./ChessGame

```

## How to play

- Enter the coordinates of the piece you want to move
- Enter the coordinates of the place you want to move the piece to
